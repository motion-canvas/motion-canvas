<IssueGroup type={'feat'}>
  <Issue user={'aarthificial'} pr={402}>
    New playback architecture:
    <ul className="margin-top--sm margin-bottom--sm">
      <li>Project settings are now stored in meta files.</li>
      <li>Preview and rendering settings are configured separately.</li>
      <li>The animation range is stored using seconds instead of frames.</li>
      <li>
        You can adjust the audio offset by holding <code>SHIFT</code> and
        dragging the waveform left and right
      </li>
      <li>
        The rendering is not locked to the framerate, making the process a bit
        faster.
      </li>
    </ul>
    This change also opens the door to a lot of new features, including the{' '}
    <Link to="https://github.com/motion-canvas/motion-canvas/issues/213">
      Presentation Mode
    </Link>
    ,{' '}
    <Link to="https://github.com/motion-canvas/motion-canvas/issues/170">
      Editable Signals
    </Link>{' '}
    and{' '}
    <Link to="https://github.com/motion-canvas/motion-canvas/issues/364">
      Custom Exporters
    </Link>
    .
  </Issue>
